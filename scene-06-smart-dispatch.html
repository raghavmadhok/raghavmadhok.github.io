T1 Surface Area Documentation

Component Overview

The T1 Surface Area is a reusable homepage component that serves as the primary interface for T1 interactions on dispatcher-facing dashboards. This same component is used across multiple scenes to surface different types of proactive or responsive information.

Smart Dispatch (Scene 6) — DECIDE Mode

In this scene, the T1 Surface Area presents a tech assignment recommendation for an unassigned RO. The dispatcher has three clear options:

  • Assign the recommended tech (primary action, blue button)
  • Choose an alternative (secondary options)
  • Modify the decision (split jobs, delay, ask for more options)

Surface Area Variants

DECIDE (Scene 6): Tech recommendations, job assignments, priority decisions
INFORM (Scene 12): Risk alerts, pattern warnings, proactive notifications
ASK (Scene 3): Query interface, "ask T1 for..." entry point

Scoring Formula Rationale

The "Why Marcus?" expandable section shows the breakdown of how T1 scored the recommendations. This transparency is critical for:

  • Dispatcher trust in the algorithm
  • Quick verification of recommendation logic
  • Easy identification of edge cases
  • Audit trail for compliance

Assigned State Transition

When a dispatcher clicks "Assign Marcus," the component could:

  • Show success confirmation (green banner with check icon)
  • Move the assigned RO to a "history" section (collapsed)
  • Surface the next unassigned RO with a new recommendation
  • Fade or dim the surface area briefly before refreshing

Key Design Patterns

  • T1 gradient top border (3px) identifies all T1-driven surfaces
  • Mode badge (DECIDE/INFORM/ASK) clarifies the interaction pattern
  • Ranked cards with "Best Match" highlight for quick scanning
  • Skill checkmarks show qualification match at a glance
  • Quick actions below provide escape routes without losing context

Responsive Considerations

The tech card grid (3 columns) collapses to 2 columns on tablet and 1 column on mobile. The scoring rationale should remain expandable but offscreen as a sheet on mobile.
